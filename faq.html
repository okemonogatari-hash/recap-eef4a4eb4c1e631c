「スキル」って結局なんですか？

💬 当日のご発言：「スキルズっていうものも概念もまだはっきりわかってない状態なんですけど…」（00:59頃）
A. ANSWER

スキル＝AI（Claude Code）に「これお願いね」と言える、自分専用の作業手順書のようなものです。

イメージで言うと…

「議事録を作る時はね、こうやって、こう書いて、ここに保存してね」っていう手順を、毎回AIに説明するのは大変ですよね。

そこで、その手順を1度だけまとめてAIに覚えさせると、次からは「議事録作って」と一言言うだけで、AIが手順通りに動いてくれる。

この「覚えさせた手順」のことを「スキル」と呼びます。

くぅさまの場合、こんなスキルが作れそうです

  • 📝 議事録作成スキル：録音 → 文字起こし → 要約 → ファイル保存
  • 📊 制服在庫集約スキル：50店舗のスプシ → マスターに自動転記
  • 📅 シフト管理スキル：シフト希望集約 → 自動配置 → 通知
  • 🏪 旦那さま用 店長業務スキル：定型作業をテンプレ化
💡 スキルの作り方（お任せください）： スキルを作るのは、おけもんカンパニー側の仕事です。くぅさまから「こんな手順でやってます」と教えていただければ、こちらでスキルとしてAIに登録するところまでサポートします🤝

会議録（議事録）ってどうやって自動で作るんですか？

💬 当日のご発言：「ここまで録音したやつをここに入れ込んだらコピペするだけはめちゃくちゃ楽ですよね」（70分台）
A. ANSWER

くぅさまがすでに半分体験されている通りで、ざっくり4ステップです。

会議録自動化のステップ

  • ① 録音する：ZoomやMeet会議をそのまま録音、または対面会議をスマホ・専用アプリで録音
  • ② 文字起こしする：録音ファイルをAIに渡す → 数分〜数十分で文字起こしテキストが出てくる
  • ③ AIに要約してもらう：文字起こしを「要約して」とAIに頼むと、決まった構成（議題・決定事項・次のアクション等）でまとめてくれる
  • ④ 自動でファイルに保存：作成した議事録を Googleドライブなどに自動保存

本日の振り返りも、実はこれで作りました

当日Circlebackというアプリで録音 → AIで文字起こし → 要約 → このページに貼り付け、という流れで作っています。本日の振り返り資料は、まさにこの仕組みのアウトプット見本でもあります✨

📌 ご準備が必要なもの： 最初の1回だけ、録音アプリ・AI・保存先（Googleドライブ等）の連携設定が必要です。設定はこちらでサポートしますので、ご安心ください。

AIに「役割」を与えて働いてもらうって、どういうこと？

💬 関連発言：「自分の範囲内で使おうとしても、でもやっぱりこの子が近くにいないと難しいですもんね」（55:00頃）
A. ANSWER

これ、本当に面白い使い方なんです🌟

AIには「あなたは経理担当です」「あなたは営業担当です」のように、役割を与えて使うことができるんです。

例えば、おけもんカンパニーの場合（社内事例として）

うちでは、1人のおけもん（おけちゃん）の中に、AIで作った仲間に役割を与えて使い分けています👇

  • 👔 技術・システムまわり担当
  • 💰 経理・お金まわり担当
  • 🔥 営業・お客様対応担当
  • ✨ 発信・マーケティング担当
  • 🌱 商品・コーチング担当
  • 🎤 全体まとめ役

1つ1つは同じClaude Codeなんですが、それぞれに「あなたは経理担当として、こういう視点で答えてください」と役割を与えているので、まるで社員が何人もいるように使えるんです。

もう1つのスタイル：「学長スタイル」（仮）

これとは別に、「1つのチャットの中で複数の役割を渡して、そのチャットの限界まで動かす」使い方もあります。

たとえば「あなたは今からPMとして〇〇して、次にデザイナーとして△△して、最後にレビュワーとして□□してください」のように、1つのチャットの中でロールを連続で切り替えていく方式。

どちらも一長一短で、業務によって使い分けます。

くぅさまの場合、こんな仲間が作れそうです

  • 👔 制服在庫担当くん：50店舗の集約だけを任せる専門家
  • 📝 議事録担当ちゃん：録音から議事録作成だけを担当
  • 📅 シフト管理さん：シフト調整専門
  • 🏪 店長補佐くん：旦那さま用の業務サポート

必要な時に必要な担当を呼ぶ → くぅさまは「監督」の立場で全体を見られる、というイメージです🎬

💡 ポイント： 役割を与える＝AIに「あなたはこういう仕事をする人だよ」と伝えるだけ。プログラミングは不要です。文章で「〇〇担当として、こんな仕事をしてください」と書くだけでOK✨

Claude CodeからAPI経由で、ChatGPT（Image-2など）やGeminiを呼び出して使えるの？

💬 当日のご発言：「黒子ちゃんとチャピチャピをどうやってつなげるのみたいな…」（51:36頃）／本来のご意図：APIでつなぐ話
A. ANSWER

結論からお伝えすると、はい、技術的には可能です✨

たとえば、Claude Codeから OpenAIのImage-2（画像生成）を呼び出して画像を作る、Claude Codeから Gemini API を呼び出してGoogleの最新AIに処理を任せる、といった連携ができます。

ざっくりの仕組み

  • ① Claude Code が「司令塔」として全体を進める
  • ② 画像が必要な箇所では、OpenAIのAPI（Image-2等）に「こんな画像お願い」と依頼
  • ③ Geminiの方が得意な処理は、Gemini APIに依頼
  • ④ それぞれから返ってきた結果を、Claude Codeがまとめて1つの成果物にする
📝 ただし、ここは話が多岐にわたります。 各APIキーの取得方法・課金プランの選び方・セキュリティ管理など、1ページではお伝えしきれない領域です。

ご興味あれば 別途ご質問ください🌷 個別にご説明させていただきます。
現時点では、Claude Code単体で十分カバーできる範囲（制服在庫管理・議事録自動化）から進めるのがオススメです。

とはいえ、覚えておくと将来便利なこと

  • 🔑 OpenAI API キーを取得すれば、Claude CodeからChatGPTやImage-2を呼び出せる
  • 🔑 Gemini API キーを取得すれば、Claude CodeからGeminiを呼び出せる（取得方法は こちら）
  • 📌 ゆくゆく社内提案で「画像生成も自動化したい」となった時、ここの仕組みが効いてきます

AIへの「指示の出し方」がよく分かりません

💬 当日のご発言：「一回やってみたんですけどやっぱりうまくいかなくて、そこでもう終わってしまって、どうやって修正をかけたらいいのか…」（40:07頃）
A. ANSWER

これは多くの方が「最初の壁」と感じる部分です。コツは3つだけ👇

コツ① 「やりたいこと」を文章で書く

プログラミング言語は不要。普段の日本語で「こうしてほしい」を書くだけでOK。

悪い例：aggregate stores（英語の単語だけ）

良い例：50店舗のスプレッドシートのデータを、マスターのスプシに集めて転記してください。サイズと枚数の合計が合わない場合は教えてください

コツ② 「うまくいかなかった時」もそのまま伝える

「さっきの方法だと、〇〇のところがうまくいかなかった。違うやり方で試してくれる？」と普通に会話する感じで伝えると、AIが別の方法を提案してくれます。

コツ③ 「分からない」と素直に言ってOK

「専門用語難しいから、もっと簡単に教えて」「初心者なので画像つきで」と書けば、AIはそれに合わせてくれます。

🤝 ご提案： 最初の数回はおけもんと一緒に画面共有しながら触っていただくのが一番です。コツが掴めると、3〜4回目には自分で進められるようになります✨

業務PCにClaude Codeを入れられない場合、どうしたらいい？

💬 関連トピック：会社のセキュリティ事情でPCに新しいソフトが入れられない可能性
A. ANSWER

これは現実的によくあるご事情です。解決策が3つあります👇

パターン①：個人PC・自宅PCで動かす

自宅のパソコンや個人スマホでClaude Codeを動かして、結果（マスタースプシ）だけを業務PCで見る形。会社のPCには何もインストール不要。

パターン②：社内提案で「新ツール導入」の許可をもらう

「これだけの業務効率化が見込めます」を提案書（本日お送りしている提案書）でお見せして、正式に導入許可をもらう。社内提案にはこちらの提案書をそのまま使ってOK。

💡 まずはパターン①から試すのがオススメです。 個人PCで動かす環境を整えれば、社内検証→効果実証→社内導入提案、という段階的な進め方ができます。

AI担当が自分一人になりそうで、ちょっと不安です

💬 当日のご発言：「会社がやっぱりAI導入とかも自分だけの領域ですることにはなるんですか」
A. ANSWER

そのお気持ち、すごく自然です。これはくぅさま個人の問題ではなく、多くの会社で同じことが起きています。

大事なのは、属人化させないこと

「くぅさんしか分からない」状態は危険なので、おけもんカンパニーでは下記をセットでお渡しします👇

  • 📖 使い方マニュアル：誰でも見れば使える状態に
  • 🎬 動画レッスン：必要に応じて操作を録画
  • 👥 後輩や同僚への引き継ぎサポート：オンライン研修も可能
  • 📞 困った時の相談窓口：おけもんカンパニーが「裏側のサポーター」

むしろ、これがチャンスでもあります

「会社で初めてAIを導入した人」というポジションは、これからの時代でもの凄く価値があります。くぅさまがその第一人者になれるよう、おけもんカンパニーが全力で支えます🤝

🌷 ご相談だけでも大歓迎です。 「ここちょっと不安」「これって普通？」という日々のご相談も、いつでもお気軽にどうぞ。一人で抱え込まないで、ご一緒に進めましょう。

やりたいこといっぱいあるけど、何から始めたらいい？

💬 当日のご発言：「やりたいこといっぱい言いすぎるんですけど何から…」（02:13頃）
A. ANSWER

「やりたいことが沢山ある」のは素晴らしいことです✨ 全部叶えられるように、優先度をご提案します👇

おすすめの順番（おけもん側のオススメ）

  • ① 制服在庫管理の自動化（最初）
    → 一番痛みが大きく、効果がすぐ見える。社内提案にも使える成功事例になる
  • ② 議事録の自動化（次に）
    → 制服在庫が回り出したら、すぐにこちらも導入。同じClaude Codeで実装可能
  • ③ シフト管理の自動化（その次）
    → 関わる人数が多いので、①②で慣れてから取り組むのが安心
  • ④ 旦那さま用 店長業務テンプレ（並行可能）
    → 規模が小さいので、いつでも着手できる。家族で実証してから会社に展開できる強み
🎯 ポイント： 全部一度にやらず、1つずつ「動く状態」にして社内に見せるのが、信頼を積み上げる最短ルートです🚀

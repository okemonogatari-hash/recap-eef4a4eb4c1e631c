📌 こちらの方法（APIキー方式）

Gemini AI（画像生成・高度なAI処理）を使いたいときの方法。スプシ操作には不向きだが、AI機能をフル活用できる。

OVERVIEWこの方法でできること

Google AI Studio（Googleが提供しているAI機能の開発スペース）からAPIキーと呼ばれる「合言葉」を取得して、自分のパソコンやアプリでGemini（Googleの最新AI）を使えるようにする方法です。

主に、こんな場面で必要になります👇

  • 🎨 AIで画像を生成したい（提案資料の挿絵・SNSの投稿画像など）
  • 📝 大量の文章をAIで処理したい（議事録の自動要約・データ整形）
  • 🤖 自動化スクリプトの中でAIを呼びたい（在庫データの異常値検出など）
💡 「APIキー」って？ 自分専用の合言葉のようなもの。これをAIに見せると「あ、この人だな」と認識してもらえて、AIの機能を使えるようになります。

REQUIREMENTS事前に準備するもの

  • Googleアカウント（普段お使いのGmailアドレスでOK）
  • パソコンとブラウザ（Chrome推奨）
  • 15分程度の時間（初回のみ）
  • 無料枠で十分始められます。クレジットカードは原則不要です
💡 無料枠の話： Geminiの無料枠は1日あたり一定回数まで使えます。個人利用やテスト目的なら、無料枠で全然足ります。たくさん使うようになったら、その時点で有料プランに切り替えるかご相談しましょう。

STEPSセットアップの流れ

Google AI Studio にアクセス

下のリンクから、Google AI Studio のサイトに行きます。

Google AI Studio APIキー画面を開く

普段使っているGoogleアカウントでサインインを求められたらサインインしてください。

「APIキーを作成」ボタンをクリック

画面に 「Create API key」 または 「APIキーを作成」 というボタンがあるので、クリックします。

画面イメージ
[Google AI Studio]
┌─────────────────────┐
│ 🔑 APIキー一覧 │
│ ───────────────── │
│ + Create API key │ ← これをクリック
└─────────────────────┘

プロジェクトを選ぶ（または新規作成）

「Google Cloud Project」を選ぶ画面が出ます。すでにプロジェクトがあれば既存のもの、なければ「新規作成」を選んでOK。

💡 補足： Google Cloud Project とは、Googleが提供しているサービスをまとめて管理する「フォルダのようなもの」です。深い理解は不要。「とりあえず1個作る」で進めてOKです。

APIキーが表示される（重要：保存する）

APIキーが画面に表示されます。これは「AIzaSyxxxxxx...」のような長い文字列です。

表示例
AIzaSyDhX3****************P66TbH9bJXrN8v4

このキーを必ずコピーして、安全な場所に保存してください。

🚨 超重要：APIキーは絶対に他人に見せない

APIキーは「自分専用の合言葉」なので、他人に見られると以下のリスクがあります：

  • 勝手にAIを使われて、料金が発生する可能性
  • 個人情報が抜かれる可能性
  • SNSや公開サイトに貼り付けない
  • メールやLINEで人に送るのも避ける

「キーを制限する」設定を必ずする（推奨）

APIキー作成画面で、「キーを制限」または「Edit API key」のリンクを見つけてクリック。

設定画面で、「APIの制限」を選び、使うAPI（例：「Generative Language API」）だけにチェック。

⚠️ なぜ制限するの？ 万が一APIキーが漏れた時、制限していないと他のサービス（Mapsなど）も勝手に使われて多額の請求が発生する事故が起きています。最初に必ず制限してください。

パソコンに「環境変数」として保存

取得したAPIキーを、パソコンの「環境変数」として保存します。

ざっくり言うと「コードの中に直接書かず、パソコンの設定ファイルに書く」ことで、安全に使う仕組みです。

  • Macの場合：ターミナルで ~/.zshenv ファイルに書く
  • Windowsの場合：システム環境変数の設定画面で追加
🤝 ご安心ください： ここはおけもんカンパニーが画面共有で一緒に進めます。コピペするだけで終わります✨

動作確認

パソコンのターミナルやアプリで、以下のように動作テストします（おけもんカンパニーがサポート）：

テスト例
$ echo $GEMINI_API_KEY
AIzaSy****************
↓
AIに「こんにちは」と話しかけて、返事が来たら成功🎉

CAUTION運用時の注意事項

① キーの管理

  • APIキーは環境変数経由でのみ使う
  • SNS・GitHub・公開サイトに絶対に貼らない
  • 定期的に「使用状況」を確認する（Google AI Studio画面で見れる）

② 漏れた時の対処

  • すぐにGoogle AI Studioでそのキーを削除して、新しいキーを発行
  • 影響範囲を確認（おけもんカンパニーが対応）

③ 課金が発生した場合

  • 無料枠を超えると課金されるので、使用量モニタリングを有効にする
  • 不安な場合は、有料アップグレードを明示的に行わなければ無料枠で停止する設定にする
🤝 ご相談： APIキーまわりは慎重に進める領域なので、初期設定は必ずおけもんカンパニーが画面共有で一緒に進めます。一人で全部やる必要はありません。
